# 광성정밀 AI-MES 시스템 - 프로젝트 개요 (Project Overview)

> 작성자: CTO Lead (Orchestrator) | 작성일: 2026-05-11 | 문서 버전: v1.0
> 분류: PDCA - Plan Phase - Overview

---

## 1. 프로젝트 비전 (Vision)

광성정밀(주)의 프레스 금속 정밀가공 라인을 **데이터 기반의 자율적 의사결정이 가능한 제조 AI 스마트공장**으로 전환한다.
단순한 MES(Manufacturing Execution System)를 넘어, IoT 실시간 데이터 + AI 예지보전 + LLM 기반 AI Agent 가 결합된 **차세대 제조 인텔리전스 플랫폼**을 구축한다.

### 핵심 슬로건
> "Press to Predict — 프레스 1타마다 데이터가 쌓이고, AI가 다음 한 타를 예측한다."

---

## 2. 사업 배경 (Background)

| 항목 | 내용 |
|------|------|
| 회사명 | 광성정밀(주) (Kwangsung Precision Co., Ltd.) |
| 주요 업종 | 자동차 부품용 프레스 금속 정밀가공 |
| 주요 고객사 | 현대/기아 1차 협력사 (Tier-1) 다수 |
| 현재 과제 | (1) 수기 작성 작업일지 및 LOT 추적 불완전 (2) 프레스 설비 돌발고장 (3) 불량 원인 추적 지연 (4) 납기 지연 리스크 가시화 부재 |
| 추진 동기 | 정부 스마트공장 고도화 사업 + 모회사 품질 요구 수준 상승 (PPAP/IATF 16949) |

## 3. 프로젝트 목표 (Objectives)

### 3.1 정량 목표 (Quantitative KPI)
| 영역 | 현재 (As-Is) | 1년 후 목표 (To-Be) | 측정 방법 |
|------|--------------|---------------------|-----------|
| 설비 종합효율 (OEE) | 65% | 82% 이상 | MES 자동 계산 |
| 불량률 (PPM) | 3,500 PPM | 800 PPM 이하 | SPC 차트 |
| LOT 추적 소요시간 | 평균 4시간 (수기) | 30초 이내 (자동) | 시스템 응답시간 |
| 설비 돌발 다운타임 | 월 12시간 | 월 3시간 이하 | 예지보전 AI |
| 납기 준수율 | 92% | 99% 이상 | 출하 데이터 |
| 일일 수기 데이터 입력 시간 | 작업자 1인당 90분 | 10분 이하 | 사용성 측정 |

### 3.2 정성 목표 (Qualitative Goals)
- 자연어로 "오늘 PRESS-03 불량률 알려줘"라고 물으면 AI Agent가 답변
- IATF 16949 / VDA 6.3 심사 시 5분 내 LOT 역추적 시연 가능
- 신입 작업자도 태블릿 UI로 30분 OJT 후 작업 가능

## 4. 프로젝트 범위 (Scope)

### 4.1 In-Scope (포함)
사업계획서 p41 솔루션 구성도 기준 **5개 핵심 모듈 + 5개 지원 모듈** 전체:

**핵심 업무 모듈**
1. AI 대시보드 (생산/품질/설비/출하 통합)
2. 원자재 입고관리 (LOT, 공급처 품질 분석)
3. 프레스 공정관리 (전용 모듈 - 핵심) — 진동/온도/네트워크/전력 IoT
4. 일반 공정관리 (후처리/조립 등 비프레스 공정)
5. 출하관리 (납기 추적, LOT 역추적)

**지원/관리 모듈**
6. 사용자/시스템관리
7. 기준정보관리 (품질기준/작업표준/코드)
8. 데이터관리 (통합 수집/시각화/AI 학습 데이터셋)
9. AI Agent 관리 (LLM 기반 통합 질의)
10. KPI 관리

### 4.2 Out-of-Scope (제외)
| 제외 항목 | 사유 |
|-----------|------|
| ERP 회계/인사 모듈 | 기존 더존 ERP 유지 (인터페이스만 구현) |
| CRM | 영업팀 별도 SaaS 사용 |
| 금형 설계 CAD 연동 | 2차 사업 (2027) |
| 협력사 포털 (외부 공급사 직접 접속) | Phase 2 (2026 Q4) |

## 6. 일정 (High-Level Schedule)

총 사업기간: **2026년 6월 ~ 2027년 5월 (12개월)**

| Phase | 기간 | 주요 산출물 |
|-------|------|-------------|
| **Phase 0: Plan** | 2026.06 (1개월) | 본 문서 세트 (Plan + Design) |
| **Phase 1: Foundation** | 2026.07 ~ 2026.08 (2개월) | 인프라 구축, IoT 게이트웨이, DB 스키마, 기준정보 |
| **Phase 2: Core MES** | 2026.09 ~ 2026.11 (3개월) | 입고/공정/출하 관리, LOT 추적, 기본 대시보드 |
| **Phase 3: AI Layer** | 2026.12 ~ 2027.02 (3개월) | 예지보전 AI, 품질 이상탐지, AI Agent (LLM) |
| **Phase 4: Integration & Test** | 2027.03 ~ 2027.04 (2개월) | ERP 연동, 통합테스트, 사용자 교육 |
| **Phase 5: Go-Live** | 2027.05 (1개월) | 안정화, 인수인계, 1차 사업 종료 |

마일스톤:
- **M1** (2026-08-31): IoT 게이트웨이로 프레스 1호기 실시간 데이터 수집 데모
- **M2** (2026-11-30): LOT 역추적 30초 데모 (모회사 SQA 참관)
- **M3** (2027-02-28): 예지보전 AI 1차 모델 정확도 ≥ 85%
- **M4** (2027-05-15): 전사 Go-Live 및 KPI 측정 시작

## 9. 성공 기준 (Success Criteria)

본 프로젝트는 다음을 모두 충족해야 **성공**으로 간주한다.

1. 사업계획서 p41-43의 10개 모듈 모두 정상 동작
2. 정량 KPI 항목 6개 중 5개 이상 목표 달성
3. 모회사 SQA로부터 LOT 추적성 인증 획득
4. 작업자 만족도 조사 80% 이상 ("기존 수기보다 편하다")
5. 정부 평가 시 사업계획 부합도 ≥ 90%
